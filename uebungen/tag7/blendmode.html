CSS Blend Modes

normal

No blending occurs; the top layer completely obscures the bottom layer.

color

Multiplies the colors of the top layer with the base layer, resulting in a blended color.

color-burn

Darkens the base layer to reflect the blend layer's colors, resulting in a darker image.

color-doge

Brightens the base layer to reflect the blend layer's colors, resulting in a lighter image.

darken

Selects the darker color from either the base or blend layer for each pixel.

difference

Subtracts the blend layer's color from the base layer's color or vice versa, resulting in a high-contrast image.

exclusion

Similar to the Difference mode but with less contrast, often used for special effects.

hard-light

Multiplies or screens the colors, depending on the blend color's lightness, often resulting in intense contrast.

hue

Retains the luminance and saturation of the base layer while adopting the hue of the blend layer.

lighten

Selects the lighter color from either the base or blend layer for each pixel.

luminosity

Retains the hue and saturation of the base layer while adopting the luminance of the blend layer.

multiply

Multiplies the colors of the base and blend layers, resulting in a darker color.

overlay

Combines Multiply and Screen blend modes, enhancing contrast while preserving highlights and shadows.

saturation

Retains the hue and luminance of the base layer while adopting the saturation of the blend layer.

screen

Inverts the colors of the base and blend layers, multiplies them, and inverts the result, resulting in a lighter color.

soft-light

This is a variation of hard-light. Similar to Overlay but with a softer effect, often used for subtle adjustments.
